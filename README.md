# Ai Tools Bolt

An Adobe Illustrator Extension built with Bolt CEP.

**Ai Tools Bolt** is an extension built to simplify common tasks within Adobe Illustrator.

Features:

- Registration Marks
- Laser File Setup
- Layer/Object Renaming
- SignAgent™️ Tools
